# Code of conduct

This project is intended to be a welcoming, practical place to collaborate.

## Expected behavior

- Be respectful and assume good intent.
- Discuss ideas and code, not personal characteristics.
- Give specific, constructive feedback.
- Respect privacy and never repost user messages, recordings, credentials, or logs.
- Accept maintainer decisions gracefully, even when you disagree.

Harassment, threats, discrimination, doxxing, spam, and publishing private information are not
acceptable. Maintainers may edit, hide, or remove content and temporarily or permanently
restrict participation when needed to protect the community.

Report conduct concerns privately through the maintainer's GitHub profile. Include links or
screenshots only when necessary and remove unrelated personal information.
