# Support

- Use a **bug report** for reproducible defects.
- Use a **feature request** for product ideas.
- Use GitHub Discussions for setup questions and general help when Discussions are enabled.
- Report security problems privately as described in [SECURITY.md](SECURITY.md).

Before asking for help, check the README, search existing issues, and include the deployment
method, Python/Docker version, relevant sanitized logs, and exact reproduction steps.

Never post real Telegram tokens, Gemini API keys, private voice messages, or production
database files.
